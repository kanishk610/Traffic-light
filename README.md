# 🚦 Traffic Light System

A beginner-friendly Traffic Light Simulation project built using **HTML, CSS, and JavaScript**.

This project simulates a real-world traffic signal system with automatic light transitions, pedestrian crossing functionality, dark/light mode, and interactive controls.

---

## ✨ Features

- 🚦 Automatic Traffic Light Sequence
- ▶️ Start Traffic System
- ⏹️ Stop Traffic System
- 🚶 Pedestrian Crossing Button
- 🌙 Dark/Light Mode Toggle
- 💡 Glowing Traffic Light Effects
- 🎨 Responsive and Interactive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

---

## 📚 Concepts Practiced

- DOM Manipulation
- Event Listeners
- `setInterval()`
- `setTimeout()`
- Array Handling
- Functions
- UI Styling & Animations
